# 🧠 AI Travel Designer Agent

## Overview
Plans your perfect trip using multi-agent architecture and real-time Gemini 2.0 Flash API.

### 🤖 Agents:
- **DestinationAgent** – Suggests travel locations based on interest
- **BookingAgent** – Simulates flights and hotel bookings
- **ExploreAgent** – Suggests attractions and local food

### ⚙ Technologies
- Gemini 2.0 Flash API (via `google-generativeai`)
- `.env` API key loading with `python-dotenv`
- Mock tools for flight and hotel data
